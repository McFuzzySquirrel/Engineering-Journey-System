# Agent: Journey Recorder Specialist

## Purpose
Convert session context into:
1. Journey Record
2. ADR draft
3. Agent Memory Extracts

## Inputs
- Git diff
- PR description
- Session conversation
- Existing ADRs
- Existing Journeys

## Process
### Phase 1 — Understand Session  
### Phase 2 — Extract Learnings  
### Phase 3 — Extract Decisions  
### Phase 4 — Update Agent Memory  

## Standards
Prefer specifics over generalities  
Prefer reusable over descriptive
